Если пришли не настроенные сканнеры ШК MINDEO MD6600-HD, необходимо отсканировать следующие ШК из ЭТОЙ инструкции.

1. Страница 18 USB device type затем USB virtual COM. Второй код дважды.

2. Cтраница 34 Truncation/Expantion затем Expand to EAN-13 дважды.

3. Cтраница 36 Truncation/Expantion затем Expand to EAN-13 дважды.

4. Страница 76 Suffix string setting (второй сверху), затем стр. 89 ШК 0, D, END.

Если надо сбросить в дефолт то смотри ЗДЕСЬ


1. Сканер сразу работает в режиме эмуляции клавиатуры (в этом режиме драйвер не требуетcя), но нам этот режим не подходит, поэтому нам необходимо сканер перепрограммировать в режим эмуляции COM, для этого нужно сосканировать ШК на стр.12 (№ страницы написан внизу слева) select usb-com-std.

2. Если сканер уже подключен к компьютеру, то тут же на экране появится надпись "Найдено новое оборудование". Вот тут надо установить драйвер, просто запустить файл R96-9636 из ЭТОЙ папки если у вас 32 битная ОС или из ЭТОЙ папки если у вас 64 битная OC.

3. Нужно сосканировать ШК на стр 102 (№ страницы написан внизу справа), а именно:

-зайти в режим программирования сосканировать ШК ENTER/EXIT PROGRAMMING MODE

-сосканировать ШК GS1-128 AIM ID=DISABLE

-выйти из режима программирования сосканировать ШК ENTER/EXIT PROGRAMMING MODE

4. Нужно сосканировать ШК на стр 125 (№ страницы написан внизу справа), а именно

-зайти в режим программирования сосканировать ШК ENTER/EXIT PROGRAMMING MODE

-сосканировать ШК UPC-A to EAN13=expand

-выйти из режима программирования сосканировать ШК ENTER/EXIT PROGRAMMING MODE

5. Нужно сосканировать ШК на стр 128 (№ страницы написан внизу справа), а именно

-зайти в режим программирования сосканировать ШК ENTER/EXIT PROGRAMMING MODE

-сосканировать ШК UPC-E to EAN13=expand

-выйти из режима программирования сосканировать ШК ENTER/EXIT PROGRAMMING MODE

P.S иногда надо сканировать ШК которые называются minimum reads для ean13 (стр 135), ean8 (стр 138), upc-a (стр 126), upc-e (стр 125), стр 143 - global settings. Это если неправильно сканируются ШК и пользователь жалуется что это часто происходит, то как вариант попробовать запрограммировать эти шк, по умолчанию там стоят значения 1 или 2, надо поставить 4
